# SAFAR Travel Booking Website

A full-stack travel booking website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

## Features
- User registration and login
- Hotel booking
- Trip management
- Owner dashboard
- Add and manage hotels
- Secure authentication
- Bargaining system for price negotiation
- Direct communication between tourist and hotel owner

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
